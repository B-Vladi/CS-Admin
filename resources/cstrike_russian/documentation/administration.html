Документация: администрирование
Администрирование сервера осуществляется через GUI-меню (Graphic User Interface), которое при стандартных настройках отображается после нажатия на клавишу "H".
Изменить клавишу можно в настройках игры (Options - Keyboard - Activate in-game GUI) либо через конфигурационные файлы (.cfg, .rc), используя команду +commandmenu.
Пример настройки:
bind "kp_enter" "+commandmenu";
Пункты меню:
  • Админцентр - основные меню администрирования сервера:

    • Забанить - показать меню бана игроков.
      Пункт Назначение
      1-6 Забанить игрока с соответствующим ником.
      7 Изменить причину бана. Доступные причины: "Читер", "Спамер", "Лагер", "Флудер", "Неадекват".
      8 Изменить время бана. Доступные значения: "Бан НАВСЕГДА", "Бан на 10 минут", "Бан на 30 минут", "Бан на 60 минут", "Бан на 120 минут".
      9 Переключить на следующих игроков.
      0 Закрыть меню.
      Для того чтобы снять бан с игрока, необходимо набрать в консоли:
      amx_unbancs "ник игрока"
      , затем войти в Web-панель администрирования и удалить бан там.

    • Кикнуть - показать меню кика игрока.
      Пункт Назначение
      1-8 Кикнуть игрока с соответствующим ником.
      9 Переключить на следующих игроков.
      0 Закрыть меню.
    • Ударить/Убить - показать меню наказания игрока (убийство или удар).
      Пункт Назначение
      1-7 Применить к игроку с соответствующим ником выбранное наказание.
      8 Изменить тип наказания. Доступные типы: "Убить", "Стукнуть на 0 HP", "Стукнуть на 1 HP", "Стукнуть на 5 HP". Удары отнимают у выбранного игрока соответствующее количество процентов от здоровья.
      9 Переключить на следующих игроков.
      0 Закрыть меню.
    • AMX Fuckoff - показать меню сильных наказаний для игроков.
      Пункт Назначение
      1-7 Применить выбранное наказание к игроку с соответствующим ником.
      8 Изменить тип наказания. Доступные наказания: "Самоубийство" (изменяет все клавиши так, что при нажатии на них, выбранный игрок умирает), "Сбить настройки" (изменяет настройки выбранного игрока, задавая им случайные значения), "Создать лаги" (понижает fps и повышает пинг у выбранного игрока), "Кружить" (заставляет выбранного игрока кружиться вокруг своей оси), "Летать" (заставляет выбранного игрока летать по карте без возможности управления передвижением), "Забить HDD" (заставляет игрока постоянно делать снимки с экрана до тех пор, пока у него не останется места на жестком диске), "Всё вместе" (применяет к выбранному игроку все предыдущие наказания), "Отменить всё" (отменяет все наказания для выбранного игрока).
      9 Переключить на следующих игроков.
      0 Закрыть меню.
    • Сменить карту - показать меню смены карты.
      Пункт Назначение
      1-8 Изменить соответствующую карту на сервере.
      9 Переключить на следующие карты.
      0 Закрыть меню.
    • Голосование за карту - показать меню голосования за карту.
      Пункт Назначение
      1-7 Добавить соответствующие карты в список для голосования. Максимальное количество карт в списке - 4, минимальное - 1.
      8 Запустить голосование из выбранных карт.
      9 Переключить на следующие карты.
      0 Закрыть меню.
    • Команда игрока - показать меню смены команды игроков.
      Пункт Назначение
      1-7 Переместить соответствующего игрока за выбранную команду.
      8 Изменить команду, за которую следует перемещать игроков. Доступные значения: "Переместить за TERRORIST", "Переместить за CT", "Переместить за SPECTATOR".
      9 Переключить на следующих игроков.
      0 Закрыть меню.
    • Запрет оружия - показать меню запрета покупки оружия.
      Пункт Назначение
      1-7 Включить либо выключить запрет для соответствующего оружия. Справа отображается текущий статут запрета (ВКЛ/ВЫКЛ).
      8 Сохранить настройки.
      9 Переключить на следующие оружия.
      0 Закрыть меню.
  • HLTV - команды для HLTV:

    • Информация - просмотр информации HLTV:

      • Общая - вывести в консоль общую информацию, такую как: версия, IP, FPS, сетевую статистику, количество подключенных наблюдателей и прокси.

      • Наблюдатели - вывести в консоль подробную информацию о подключенных наблюдателях.

      • Прокси - вывести в консоль подробную информацию о подключенных прокси-серверах.

    • Демо - управление демо файлами:

      • Start - начать запись демо. В правом верхнем углу отображается результат выполнения команды:
        Сообщение Значение
        Recording initialized. Запись демо началась.
        Already recording to demo/имя_файла.dem. В данный момент записывается демо с именем имя_файла.dem.
      • Stop - остановить запись демо. В правом верхнем углу отображается результат выполнения команды:
        Сообщение Значение
        Completed demo demo/имя_файла.dem. Запись демо с именем имя_файла.dem успешно завершено.
        Not recording. В данный момент демо не записывается.
        Имена для файлов создаются автоматически. Формат имени: HLTV-ггммддччмм-имя_карты.dem (последние 2 цифры года, месяц, день, часы, минуты).
        Все созданные демо доступны из корневой папки FTP-хранилища.

    • Подключить - подключить HLTV к серверу. Все подключенные наблюдатели и прокси переключатся вместе с HLTV. Записывающиеся демо так же будут переключены.

      • ClanWar - подключить к серверу cw.cs-console.ru.

      • Public - подключить к серверу cs-console.ru.

    • Остановить - остановить HLTV. Подключенные наблюдатели и прокси не будут отключены. Записывающиеся демо будут приостановлены и возобновлены при следующем подключении к серверу.

    • Отключить - отключить HLTV, сбросить всех наблюдателей и прокси, завершить запись демо.

  • Clanwar - управление ClanWar-сервером:

    • Пароль - установка пароля на сервер:

      • clanwar - установить пароль clanwar.

      • Сбросить - снять пароль с сервера.

    • Warm Up - Режим Warm Up:

      • Включить - установить режим Warm Up.

      • Выключить - снять режим Warm Up.

    • 3 Рестарта - сделать 3 рестарта карты и загрузить стандартый конфиг (отключенный режим Warm Up).

  • Информация - просмотр информации сервера:

    • Статистика - статистика игры (только для public-сервера):

      • hp, me, rank - показать текущую статистику:
        say "/hp"; say "/me"; say "/rank";
      • Игроков - показать меню просмотра статистики игроков, присутствующих на сервере.
        Пункт Назначение
        1-7 Отобразить выбранный тип статистики для соответствующего игрока.
        8 Изменить тип отображаемой статистики. Доступные типы: "Статистика на карте", "Общая статистика".
        9 Переключить на следующих игроков.
        0 Закрыть меню.
      • На карте - показать вашу статистику на карте:
        say "/statsme";
      • Общая - показать вашу общую статистику:
        say "/rankstats";
      • TOP 15 - показать 15 лучших игроков сервера:
        say "/top15";
      • Сбросить - Сбросить статистику. При выполнении этой команды автоматически перезагружается сервер.

    • Motd - показать окно приветствия, показываемое игрокам при подключении к серверу.

    • Он-лайн игроки - показать в консоли список игроков, находящихся на сервере. Список выглядит в виде таблицы, содержащей информацию о игроках: id авторизации на сервере, ник, steam id, общий id на сервере, наличие флага a, наличие флага c и права.

    • Последние игроки - вывести в консоли список нескольких последних игроков, присутствовавших на сервере. Список выглядит в виде таблицы, содержащей информацию о игроках: ник, steam id, ip-адрес и права.

  • Разное - различные команды управления сервером:

    • Послать сообщение - различные режимы посылки сообщения в чат:

      • Всем - послать сообщение для всех игроков без указания вашей команды. После выбора пункта необходимо ввести сообщение (вводимые символы отображаются в левом верхнем углу экрана), затем нажать enter для отправки или escape для отмены.

      • Админам - послать сообщение только администраторам. После выбора пункта необходимо ввести сообщение (вводимые символы отображаются в левом верхнем углу экрана), затем нажать enter для отправки или escape для отмены.

    • Рестарт - сделать рестарт:

      • Карта - карты.

      • Сервер - сервера.

    • Тэг - добавить или убрать тег команды для вашего ника.

  • Настройки - основные клиентские настройки:

    • Правая рука - устанавить оружие в правую руку.

    • Левая рука - устанавить оружие в левую руку.

  • Закрыть - закрыть меню.
